# User Guide

## Overview
Step-by-step usage for core citizen and admin flows.

## Citizen Flow

### 1. Registration
- Open `/register`.
- Fill full name, email, password, and county.
- Submit and confirm redirect to dashboard.

### 2. Submit Report
- Open `/report/new`.
- Select category, severity, and add title/description.
- Pin location and optional photos.
- Submit and note reference code.

### 3. Track Report
- Open `/my-reports`.
- Filter by status and open report details.
- Follow timeline updates and public admin notes.

### 4. Reset Password
- Open `/forgot-password`.
- Submit account email.
- Open reset link and set new password.

## Admin Flow

### 1. View Dashboard Analytics
- Open `/admin`.
- Use status/date filters.
- Review status distribution, category trends, county load.

### 2. Export Reports
- On dashboard, click `Export CSV` or `Export PDF`.
- Use date and status filters before exporting.

### 3. Update Report Workflow
- Open recent report, click `Update`.
- Change status, assign technician, add comment.
