# Help/Support Documentation

## Overview
Help channels and in-app troubleshooting for citizens and admins.

## Support Channels

- Email: `support@majiwatch.local`
- In-app FAQ pages:
  - Citizen: `/help`
  - Admin: `/admin/help`
- Incident escalation (admin): county IT desk + water authority operations lead.

## Common Issues

### Password reset email not received
- Confirm email spelling.
- Check spam/junk folder.
- Request another reset link after 1 minute.

### Cannot submit report
- Ensure location coordinates are within Kenya bounds.
- Ensure description is at least 20 characters.
- Reduce image count to 5 or fewer.
